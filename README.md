# Meijie Hu - Portfolio

A portfolio site for Meijie Hu, built with vanilla HTML, CSS, and JavaScript.

## Features

- Custom viewport-based grid system matching the original design
- Animated custom cursor with mix-blend-difference
- Project list with staggered fade-in animations
- List/Slider view toggle
- Canvas-based hover preview backgrounds
- Loading animation sequence
- Responsive layout (desktop + mobile)
- Page transition effects

## Running Locally

```bash
cd cathy-dolle-replica
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

## Tech Stack

- HTML5
- CSS3 (with Tailwind CSS via CDN)
- Vanilla JavaScript (ES6+)
- No build tools required
